***********************************************************************************
								Prueba Automatización Appium
***********************************************************************************

El proyecto consiste en una automatización de una calculadaora en la plataforma de Android 9.0,
en la cual se automatizó las operaciones básicas de suma, resta, multiplicación y división.
Para la ejecución del mismo se se debe instalar y configurar el Android Studio para emular
el dispositivo virtual.
Adicional se debe instalar y configurar el Appium Desktop.

Para la construcción del proyecto se utilizó el patrón de automatización POM,
con lenguaje de programación Java, para el manejo de las dependencias se utilizó Gradle,
para ejecutar las pruebas se utilizó JUnit, Cucumber y SerenityBDD para la generación
del reporte y se utilizó Git para el control de versiones.
